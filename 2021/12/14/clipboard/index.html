Mac's Universal Clipboard fails
Zehao Liu
 2021-12-14 00:12:58

I have lots of Apple devices, one of the most common functions I use is Universal Clipboard, which means I can copy the words in one mac and past it in other devices(iphone/ipad). But what annoys me is that it often fails, I have to write down efficacious steps I use to slove the problem, since I will meet again, again…

  • Check bluetooth open/wifi in same network/login same appleid. Note: these are base requirements, but I satisfed all in gerenal.
  • Use third device to test which is fail, for example, when find two macs can’t use universal calipboard, we can use iphone to test problem in which one.
  • *Though we can find a many methods in google, but I want to say none of them can work. one of them I think may work is open Activity Monior(can find in spotlight), and then search a process named “pboard”
  • reload the faild machine test in step 2

Note: I have bought logitech master3 mouse, which also support universal calipboard, I can’t stand this feature of APPLE anymore.
